这是一份基于视频内容的 **Fold Craft Launcher (FCL) 启动器全面教程 2.0** 详细梳理。本教程涵盖了从下载安装到联机游玩的全流程。

### 一、 启动器的下载与安装 (01:00 - 01:39)

1.  **下载版本选择：**
    *   面对不同架构的 APK，现代安卓手机（近几年的）通常首选 **`arm64-v8a`** 版本（体积小，性能好）。
    *   **备选方案：** 如果安装后首次运行遇到 JAVA 运行环境 (JRE) 安装失败，请尝试下载并安装 **`all`** 通用版本（包含多种架构的运行环境，体积较大）。
2.  **首次运行设置：**
    *   安装并打开 FCL。
    *   **授权：** 首次进入会弹窗请求存储权限，必须点击“确认”并允许管理所有文件。
    *   **协议：** 点击“同意并继续”。
    *   **环境安装：** 点击右下角的“安装/更新”按钮，下载必要的 JAVA 运行环境。

### 二、 三种账户登录详解 (01:39 - 03:50)

在主界面右侧点击“没有账户”添加账户，有三种方式：

#### 1. 离线登录 (适合无正版玩家)
*   **操作：** 点击“离线账户” -> 输入游戏 ID -> 登录。
*   **注意：** 用户名仅支持 **大小写英文字母、数字、下划线 (`_`)**。
*   **报错处理：** 如果进入游戏提示 `Invalid characters in username`，说明你用了非法字符（如中文、空格），请重新创建纯英文/数字 ID。

#### 2. 微软登录 (适合正版玩家)
*   **操作：** 选择“微软账户” -> 点击登录 -> 自动复制一段代码 -> 跳转浏览器粘贴代码 -> 登录微软账号并授权。
*   **常见报错与解决：**
    *   **无法连接认证服务器：** 通常是网络问题。尝试切换流量/WiFi，或使用网络加速工具。
    *   **微软账户没有链接到 Xbox 账户：** 意味着你从未在微软官方启动器（PC端）玩过。需要先在电脑正版启动器登录并创建一次角色档案。
    *   **账户尚未获取 Minecraft: Java Edition：** 代表你的微软账号没买游戏。如果确定买了，尝试上一步；没买则只能用离线或外置登录。

#### 3. 外置登录 (适合 LittleSkin 等皮肤站用户)
*   **添加服务器：** 点击左下角“+ 认证服务器” -> 输入皮肤站地址（例如 LittleSkin: `https://littleskin.cn/`） -> 确认。
*   **注册与设置 (以 LittleSkin 为例)：**
    1.  点击带加号的按钮打开浏览器去官网注册。
    2.  **角色管理：** 登录后进入“角色管理” -> “添加新角色”。**注意：** 角色名必须符合规范（英文/数字/下划线）。
    3.  **皮肤库：** 进入“皮肤库” -> 选择或上传皮肤 -> 点击“穿戴”应用到刚才创建的角色上。
    4.  **登录 FCL：** 回到 FCL，选择刚才添加的 LittleSkin 服务器，输入你在皮肤站注册的账号密码即可登录，此时会显示你的皮肤。

### 三、 核心界面与基础设置 (03:50 - 06:50)

点击左侧第二个按钮（扳手图标）进入设置界面：

#### 1. 游戏设置 (版本独立设置)
*   **启用游戏特定设置：** **强烈建议开启**。开启后，该版本的设置（内存、渲染器等）将独立于全局设置，互不干扰。
*   **管理版本：** 可快速定位当前版本的资源文件夹（模组、光影等路径）。
*   **自动安装：** 下载纯净版游戏后，可在此一键安装 Forge, Fabric, OptiFine 等加载器。

#### 2. 下载界面 (左侧第三个按钮)
*   功能强大，可下载：游戏版本、整合包、模组、资源包、世界存档、光影。

#### 3. 按键布局 (左侧第四个按钮)
*   可自定义按键位置。
*   **推荐：** 点击左下角“下载布局”，直接应用社区玩家分享的优秀布局（如“四指PVP”、“仿原神”等），省去从零编辑的麻烦。

#### 4. 启动器设置 (左侧第五个按钮 - 全局设置)
*   **Java 版本：** 默认“自动选择”即可。除非特定整合包要求高版本 Java。
*   **版本隔离：** **务必开启！** 防止不同版本的模组文件冲突。
*   **游戏内存：**
    *   关闭“自动分配内存”。
    *   根据手机可用内存手动分配。例如 16GB 手机，当前可用 6GB，则分配给 FCL 约 6000MB (6GB)。
    *   **注意：** 手机开启的“内存扩展/虚拟内存”FCL 无法使用，只能看物理内存。
*   **窗口分辨率：** 默认 100% (原生分辨率)。为了流畅度，建议调整为 **50%** 左右，能大幅提升帧数。
*   **渲染器选择 (关键)：**
    *   **Krypton Wrapper:** 推荐，支持全版本，内置。
    *   **Holy GL4ES:** 仅支持 `<=1.21.4` 版本。运行 Sodium/Embeddium 等优化模组时需修改配置文件。
    *   **MobileGlues (MG):** 仅支持 `>=1.17` 版本。性能最强，兼容性好，但需单独安装 App（后文详解）。
*   **启动器自身设置：**
    *   **忽略刘海屏(全屏)：** 开启可解决启动器黑边问题。
    *   **下载源：** 如果下载报错，尝试切换下载源（如 BMCLAPI）。

### 四、 游戏版本与模组加载器安装 (06:50 - 07:54)

1.  **选择版本：**
    *   模组玩家推荐模组生态丰富的版本：**1.20.1, 1.16.5, 1.12.2**。
    *   原版玩家任意选择。
2.  **理解模组加载器 (Mod Loader)：**
    *   **Forge:** 历史最久，模组数量最庞大，生态最完善。高版本逐渐被 NeoForge 取代。
    *   **Fabric:** 轻量化，启动快。大量性能优化模组（如 Sodium 钠）原生基于 Fabric。
    *   **注意：** 不同加载器的模组**互不兼容**。下载模组时必须对应（Forge 版模组只能用在 Forge 上）。
    *   **Fabric 特别注意：** 安装 Fabric 模组前，通常必须同时安装 **Fabric API**。

### 五、 模组管理：内部下载与外部导入 (07:54 - 09:58)

#### 方法一：FCL 内部安装 (推荐)
1.  进入“下载”界面 -> 点击“模组”。
2.  **筛选：** 左侧筛选出对应的游戏版本（如 1.20.1）和加载器（如 Forge）。
3.  搜索并下载模组（例如 Biomes O' Plenty）。
4.  **重要：检查前置模组 (Dependencies)。** 下载界面会提示该模组需要的前置（如 TerraBlender, GlitchCore），必须一并下载，否则游戏会崩溃。

#### 方法二：外部文件管理器导入 (以 MC百科为例)
1.  在 MC百科 (mcmod.cn) 找到模组（例如 虚无世界3 / Advent of Ascension 3）。
2.  查看“模组关系”，下载所有必要的前置模组。
3.  使用 **MT管理器** (或类似文件管理 App)。
4.  **路径：** 找到 `/storage/emulated/0/FCL/.minecraft/versions/[你的版本号]/mods`。
    *   *提示：`.minecraft` 是隐藏文件夹，需在文件管理器设置中开启“显示隐藏文件”。*
5.  将下载好的 `.jar` 模组文件移动/复制到 `mods` 文件夹内。
6.  启动游戏即可看到模组内容。

### 六、 安装 MG 渲染器和光影 (09:58 - 11:18)

#### 1. MobileGlues (MG) 渲染器配置
*   **安装：** 下载 MG 的独立 APK 并安装。
*   **授权：** 打开 MG App -> 点击“修改渲染器设置” -> 授权访问文件夹。
    *   在授权界面右上角点“+”，新建一个名为 **`MG`** 的文件夹。
    *   点击“使用此文件夹”并允许访问。
*   **参数设置 (参考视频)：**
    *   建议开启 `ANGLE` (解决渲染问题)。
    *   建议开启 `FSR` (提升画质/性能)，但如果遇到渲染异常（如屏幕上出现两个游戏画面），请关闭 FSR。
*   **FCL 设置：** 回到 FCL -> 游戏设置 -> 渲染器 -> 选择 **`MobileGlues`**。

#### 2. 光影安装教程
*   **前置模组：**
    *   **Fabric 版 (推荐 1.21.1)：** 下载并安装 **Sodium (钠)** 和 **Iris**。
    *   **Forge 版：** 下载并安装 **Embeddium** 和 **Oculus**。
*   **下载光影：** 在 FCL 下载界面 -> “光影” -> 下载喜欢的光影包 (如 BSL, Complementary)。
*   **启用：** 启动游戏 -> 视频设置 -> 光影包 -> 选择刚才下载的光影。
*   **注意：** 使用光影请务必配合 **MG 渲染器** 以获得最佳性能。

### 七、 远程联机教程 (11:18 - 12:22)

使用 FCL 自带的联机工具 **EnchantNet**。

1.  **房主操作：**
    *   打开 EnchantNet App -> 选择“我想当房主” -> 点击“开始扫描”。
    *   进入 Minecraft 游戏 -> 进入存档 -> 按 ESC -> “对局域网开放”。
    *   游戏左下角会提示端口，EnchantNet 会自动复制一个**邀请码**。
    *   将邀请码发给朋友。
2.  **访客操作：**
    *   打开 EnchantNet App -> 选择“我想当访客”。
    *   粘贴房主发来的邀请码 -> 点击“加入房间”。
    *   等待连接成功。
3.  **进入游戏：**
    *   访客进入 Minecraft -> 多人游戏 -> 此时列表中会出现“EnchantNet 联机大厅” -> 点击加入服务器即可联机。
