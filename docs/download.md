# 下载
## 下载启动器

### FCL (Fold Craft Launcher)

- [FCL 下载站](https://foldcraftlauncher.cn/down.html)  
  提供 FCL、ZL、ZL2、各种渲染器、JAVA 和驱动。
- [GitHub Releases](https://github.com/FCL-Team/FoldCraftLauncher/releases)  
  官方发布页面，包含所有历史版本。

### ZL (Zalith Launcher)

- [ZL 官网](https://www.zalithlauncher.cn/download)  
  ZL1 和 ZL2 官方下载页面。
- [ZL1 GitHub](https://github.com/ZalithLauncher/ZalithLauncher/releases)  
- [ZL2 GitHub](https://github.com/ZalithLauncher/ZalithLauncher2/releases)  

### 镜像站

- [柠枺镜像](https://mirror.lemwood.icu)  
  提供 FCL、ZL、ZL2、shizuku、MG 渲染器和 HMCL（电脑版启动器）。

## 下载工具

- [MT管理器](https://mt2.cn)  
  MT管理器不只是文件管理器，它的功能太强大了。
- [shizuku](https://mirror.lemwood.icu/download/shizuku/v13.6.0/shizuku-v13.6.0.r1086.2650830c-release.apk)  
  仅次于 root 的安卓 ADB 工具，配合 MT 管理器食用更佳。[官网地址](https://shizuku.rikka.app/zh-hans/)
- [[DH]遥远的地平线手机专用补丁版（网站不再维护）](https://dh.lemwood.cn)  
  自动注入 libzstd-jni 手机版补丁，支持 aarch64 架构，为手机端 Minecraft 提供完美支持，还有自带翻译支持。
